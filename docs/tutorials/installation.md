# Installation

## Luau

=== "wally"

    Add the following line to your `wally.toml`:

    ```toml
    Rovo = "wireofparadise/rovo@0.1.0"
    ```

    > This version may not always be up-to-date. Please make sure to check [the latest release version on GitHub](https://github.com/wireofparadise/rovo/releases/latest) and use that value as your package version.

    And then reinstall your packages, using the following command:

    ```sh
    wally install
    ```

    ???+ note "Wally package types"
        For wally, I recommend using [wally-package-types](https://github.com/JohnnyMorganz/wally-package-types) to foward exported types to the module script generated by wally. This allows you to reference Luau types used by Spaceship.

=== "rbxm"

    You can also install Spaceship as a standalone roblox model!

    Navigate to [the latest release on GitHub](https://github.com/wireofparadise/rovo/releases/latest), download the `Rovo.rbxm` or `Rovo.rbxmx` file, and insert it into your place.

## TypeScript

```sh
npm install @rbxts/rovo
```

!!! warning "JSX Usage"

    JSX is currently **not supported**, but I plan to add it in the future.
